Installation
------------

Requirements
^^^^^^^^^^^^

* `Python <https://www.python.org>`__ 3.6+
* `Huggingface Transformers <https://huggingface.co/transformers/>`__ 3.5+

Installation via Pip
^^^^^^^^^^^^^^^^^^^^

``pip install ckip-transformers``.
